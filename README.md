# blog

A blog site using a REST API to get info from the server

## Getting Started

First just

```
git clone https://github.com/joaopedrodcf/blog.git
```

Then

```
npm install
```

Finally

```
npm start
```

Now its ready to use

### Prerequisites

You need to install Node.js
